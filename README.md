Usage: 

Windows:

	x:\SteamDownload-Update.py [-h] [-f] [-d] [-p] [-v]

Unix:

	./SteamDownload-Update.py [-h] [-f] [-d] [-p] [-v]



optional arguments:

	  -h, --help  show this help message and exit
  
	  -f          Force update Mods
  
	  -d          Force update special directory content
  
	  -p          Enable pause message
  
	  -v          Verbose
